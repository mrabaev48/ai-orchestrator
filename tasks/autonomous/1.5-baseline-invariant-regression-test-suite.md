1. Feature summary
Собрать детерминированный baseline invariant regression suite как release gate для мини-последовательности 1.1–1.4.

2. Relevant modules
- `packages/core/**` (domain invariants)
- `packages/execution/**` (runtime safety behavior)
- `packages/state/**` (persistence/evidence/idempotency)
- `turbo.json` / test pipelines

3. Existing behavior
Нет единого acceptance-suite, который одновременно доказывает policy persistence, evidence integrity, dedup safety и timeout/cancel invariants.

4. Proposed design
- Создать `baseline-invariants` test suite с фиксированными deterministic fixtures.
- Покрыть обязательные сценарии:
  - success path
  - policy deny path
  - dedup suppression regression
  - timeout with safe retry
  - cancellation mid-step with consistent state
  - evidence checksum integrity violation detection
- Добавить CI-gate: suite обязателен перед phase-complete mark.
- Отчеты suite должны включать correlation identifiers для incident replay.

5. Files likely to change
- `packages/execution/src/__tests__/baseline-invariants/*.test.ts`
- `packages/state/src/__tests__/baseline-invariants/*.test.ts`
- `packages/core/src/__tests__/baseline-invariants/*.test.ts`
- `turbo.json` / package test scripts

6. Risks
- Flaky tests из-за nondeterministic timers/UUID.
- Долгое выполнение suite без правильного test partitioning.
- Ложноположительные результаты при неполной изоляции хранилища.

7. Test plan
- Suite self-test: запуск с seeded clock/random.
- Contract tests: policy/evidence/idempotency/error shapes.
- Integration tests: end-to-end invariant scenarios.
- Negative tests: intentional integrity/cancellation/timeout faults.

8. Rollout / migration notes
- Включить suite сначала как non-blocking (informational), затем переключить в blocking gate.
- Зафиксировать SLA по длительности suite и стратегию параллелизации.
- Документировать минимальные артефакты для аудита релизов.

9. Recommendation
Использовать как финальный go/no-go критерий baseline-invariants slice; без passing 1.5 phase не считается завершенной.
