1. Feature summary
Внедрить end-to-end idempotency key flow для non-idempotent side effects с dedup registry и policy/evidence linkage.

2. Relevant modules
- `packages/core/idempotency/**`
- `packages/execution/idempotency/**`
- `packages/state/idempotency/**`
- `packages/execution/git/**` (commit/push/PR guards)

3. Existing behavior
Защита от duplicate side effects неполная: ключи и dedup-проверки не гарантированно сквозные для всех критичных операций.

4. Proposed design
- Добавить canonical idempotency key builder на основе: `tenantId/projectId/runId/stepId/sideEffectType/normalizedInputHash`.
- Ввести dedup registry состояния: `pending|succeeded|failed|expired` с TTL и lock-guard.
- Перед non-idempotent действием выполнять `reserve(key)`; при duplicate — short-circuit с `dedupSuppressed=true`.
- Линковать dedup outcome к `policyDecisionId` и `evidenceId`.
- Для commit/push/PR добавить жесткий guard: нельзя выполнить action, если key уже `succeeded`.

5. Files likely to change
- `packages/core/src/idempotency/build-idempotency-key.ts`
- `packages/state/src/idempotency/dedup-registry.ts`
- `packages/execution/src/idempotency/side-effect-dedup-guard.ts`
- `packages/execution/src/git/**`

6. Risks
- Коллизии ключей из-за неполной canonicalization.
- Deadlock/lease leak в `pending` при crash.
- Некорректный TTL может открыть окно для повторных side effects.

7. Test plan
- Unit: один и тот же canonical input дает стабильный ключ.
- Unit: duplicate suppression корректно возвращает deterministic outcome.
- Unit: succeeded key блокирует повторный commit/push/PR.
- Integration: retry после частичного сбоя не дублирует внешний side effect.
- Regression: ключи различаются при реальном изменении payload.

8. Rollout / migration notes
- Ввести shadow-mode dedup (observe-only), затем enforce-mode.
- Миграция registry schema additive, с индексами по key/status/expiresAt.
- Операционный runbook для ручного unlock stale `pending`.

9. Recommendation
Запускать после 1.1 и предпочтительно после 1.2, чтобы каждое dedup решение имело полную evidence трассу.
